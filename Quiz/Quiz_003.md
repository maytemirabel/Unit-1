## Quiz 3 ##

Create a program that translate the proteins in the DNA chain
```.py
protein = input("Insert protein: ")
if protein.upper() == "A":
    print("T")

elif protein.upper() == "G":
        print("C")

elif protein.upper() == "T":
        print("A")

elif protein.upper() == "C":
        print("G")
```
