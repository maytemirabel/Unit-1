## Quiz 3
#### Program code
Create a program that translate the proteins in the DNA chain
```.py
protein = input("Insert protein: ")
if protein.upper() == "A":
    out_protein = "T"
if protein.upper() == "G":
    out_protein = "C"
if protein.upper() == "T":
    out_protein = "A"
if protein.upper() == "C":
    out_protein = "G"

print (out_protein)
```
