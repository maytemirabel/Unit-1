## Quiz 6
#### Program code
Given a string, create a program that produces the sum of the characters in the string.
```.py
string=input('input your string: ')
sum=0
for letter in string:
    sum+=ord(letter)
print(f'The sum of the ASCII code in your string is {sum}')
```
