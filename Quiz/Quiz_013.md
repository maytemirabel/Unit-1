## Quiz 13 
#### Program code
Create a function that receives one input and produces the output shown
```.py
def mystery (a:int, b: int) -> int:
    return (a*b)-abs(a-b)

print (mystery(2,6))
print (mystery(4,10))
print (mystery(10,10))
print (mystery(70,50))
```
